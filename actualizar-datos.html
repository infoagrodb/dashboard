Necesitas un token de acceso personal (fine-grained) del repositorio infoagrodb/dashboard con permiso Actions: Read and write. El token solo se usa en tu navegador para llamar a la API de GitHub y, si marcas la casilla, se guarda únicamente en localStorage de este dispositivo — nunca se envía a ningún otro sitio ni se sube al repositorio. No compartas esta página con el token guardado si el equipo es compartido.
